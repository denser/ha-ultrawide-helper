# Ultrawide Screen Helper for Home Assistant

Автоматически адаптирует карточки Lovelace для ультрашироких мониторов.

## Возможности
- ✅ Автоматическое растягивание карточек на ширину экрана
- ✅ Включение/отключение через YAML
- ✅ Не требует настройки каждой карточки
- ✅ Работает со стандартными и кастомными карточками

## Установка
1. Установите через HACS (рекомендуется)
2. Перезапустите Home Assistant
3. Добавьте карточку на дашборд:
   ```yaml
   type: custom:ultrawide-helper-card
   enabled: true
